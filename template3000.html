ℹ️ General Appearance
To create a more modern appearance, the template includes four packages. The parskip package removes paragraph indentation and adds vertical spacing between paragraphs. The lmodern package replaces the default Computer Modern font with Latin Modern, offering a broader character set and consistent rendering. The setspace package, with the onehalfspacing option, increases the line spacing to 1.5, improving readability and meeting academic or professional formatting standards. These adjustments affect spacing, font rendering, and symbol availability, giving the template a more contemporary and structured layout. The pifont package enables the use of special symbols, such as checkmarks and circled numbers.

\usepackage{parskip}

\usepackage{lmodern}

\usepackage[onehalfspacing]{setspace} %The formal requirements for theses or dissertations are often set at 1.5 line spacing

\usepackage{pifont}

ℹ️ Colors and Frames
To enhance the use of colors and framed content in the template, three key commands are included. The xcolor package with the options [table,xcdraw] enables color support for tables and provides an extended set of predefined colors. The framed package is used to create framed content, such as boxed text or highlighted sections. It works in combination with the shaded environment, which allows background shading for text blocks. The command \definecolor{shadecolor}{rgb}{0.7,0.85,1} specifically defines the shadecolor, which determines the background color used within shaded environments from the framed package. This ensures that the framed content can have a customized, colored background, making important sections more visually distinct.

\usepackage[table,xcdraw]{xcolor}

\usepackage{framed}

\definecolor{shadecolor}{rgb}{0.7,0.85,1} %light blue for shaded-environment with framed-package

ℹ️ Pagination Control
The commands \clubpenalty, \widowpenalty, and \displaywidowpenalty control pagination in LaTeX by preventing orphans (club lines) and widows in text and mathematical environments. \clubpenalty prevents a single line from appearing at the bottom of a page, while \widowpenalty prevents the last line of a paragraph from appearing alone at the top of a new page. \displaywidowpenalty applies the same rule specifically to mathematical environments (e.g., equations). Setting these values to 10000 enforces strict prevention, ensuring a more balanced and typographically clean document layout.

\clubpenalty = 10000

\widowpenalty = 10000

\displaywidowpenalty = 10000

ℹ️ Line Breaking Adjustments
\setlength{\emergencystretch}{2em} increases the flexibility of line breaking in LaTeX. It allows LaTeX to slightly stretch lines (up to 2em) when it struggles to find an optimal line break, helping to reduce Overfull \hbox warnings and improve text justification. This prevents text from extending beyond the right margin of the page, which can otherwise cause layout issues. It is particularly useful for documents with long words or justified text.

\setlength{\emergencystretch}{2em}
